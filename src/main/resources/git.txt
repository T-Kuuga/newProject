将项目通过git来进行管理
1.自己创建好基本项目结构
2.将我们的项目变成一个git仓库->git init
3.因为我们代码仓库只需要存放我们的代码,像class文件,
编译器自身生成的文件都不需要放入到仓库中,
而且如果放入还会经常出现代码冲突,解决方案就是
在我们的忽略文件 .gitignore 中设置
4.在设置好忽略文件后我们就可以将我们的代码提交到我们的本地仓库
这时候对于我们的文件我们需要先通过
git add 要添加的文件的路径  来将我们要放入本地仓库的代码
进行添加操作,接下来通过git commit -m "提交信息"
来真的将我们的项目代码提交到本地仓库,这时候才是真的将代码保存在
本地仓库,我们希望跟别人共同进行代码的管理就需要将我们
本地本地仓库和远程git仓库进行连接,并将代码推送到远程服务器上,
这样才能跟别人进行共同开发git remote add origin 远程服务器上面的地址(https://github.com/T-Kuuga/teamProject.git)
这样连接好之后我们就可以进行项目从本地到服务器的推送了git push -u origin master

合作开发的步骤
1.项目组长将框架他搭建好,上传到github
2.项目组长邀请组员成为协作者
3.组员git clone 项目的克隆拉取
4.coding完成后进行代码的上传
5.先pull拉取,将服务器上最新的代码拉取到本地
6.push推送自己的代码到服务器
